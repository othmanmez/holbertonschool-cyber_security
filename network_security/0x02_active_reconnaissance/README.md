# Active Reconnaissance – cyber_netsec_0x02

## Description
Ce projet est dédié à l’**active reconnaissance** en cybersécurité.  
Il a pour objectif de comprendre, utiliser et expliquer les principaux outils et techniques employés pour collecter activement des informations sur une cible réseau.

La cible principale de ce projet est : **cyber_netsec_0x02**.

## Learning Objectives
À la fin de ce projet, vous devez être capable d’expliquer, **sans l’aide de Google**, les concepts suivants :

- Qu’est-ce que l’**active reconnaissance** ?
- Pourquoi l’active reconnaissance est-elle importante en cybersécurité ?
- Comment **Wappalyzer** peut être utilisé pour l’active reconnaissance ?
- Qu’est-ce que la **DNS enumeration** ?
- Comment énumérer des **services SMTP** à l’aide d’outils en ligne de commande ?
- Comment effectuer un **OS fingerprinting** ?
- Qu’est-ce que **sqlmap** et comment l’utiliser ?
