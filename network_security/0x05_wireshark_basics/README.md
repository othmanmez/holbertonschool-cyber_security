- What Wireshark is

- How to analyze a packet using Wireshark  

## Requirements

### General
- Allowed editors: `vi`, `vim`, `emacs`
- All scripts will be tested on **Kali Linux**
- A `README.md` file at the root of the project folder is **mandatory**
- You are free to choose any IP address for testing

### More Info
- All files must contain **two lines**
- All files must end with a **new line**
- All Wireshark filters must be written in **.txt format**

Example:
```bash
cyber-sec@ubuntu$ cat ip_filter.txt
ip.addr==192.0.2.1
